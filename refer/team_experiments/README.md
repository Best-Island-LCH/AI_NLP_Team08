## 📝 기여 방법

### 1. 새로운 기법 추가

1. `TEMPLATE.md`를 복사
2. 파일명을 `[날짜]_[이름]_[기법명].md` 형식으로 변경
   - 예: `20260111_홍길동_LayerWise_LR.md`
3. 템플릿에 따라 내용 작성
4. PR 또는 직접 커밋

### 2. 빠른 메모 (TIL)

간단한 발견이나 팁은 아래 형식으로 이 README에 직접 추가:

```markdown
### YYYY-MM-DD (이름)
- **발견**: 간단한 설명
- **효과**: 성능 변화
- **참고**: 링크 (있으면)
```

## 📋 작성 가이드라인

### 좋은 실험 기록의 조건

1. **재현 가능**: 다른 팀원이 따라할 수 있도록 코드와 설정 포함
2. **비교 명확**: 베이스라인 대비 변화량 명시
3. **실패도 기록**: 안 된 것도 기록 (시간 낭비 방지)

### 성능 보고 형식

```
베이스라인: F1 macro 0.XX
적용 후: F1 macro 0.XX (+X.X%)

- consistency: 0.XX → 0.XX
- ...
```
